Workflow — Anyone Can Do Everything
STEP 1
Any account uploads a document. File saved on server. SHA256 hash stored on blockchain.
STEP 2
Uploader or anyone forwards the document to another wallet address.
STEP 3
Any connected account approves or rejects the document with a reason.
STEP 4
Anyone marks the document as Verified after approval. Recorded permanently.
STEP 5
Anyone uploads any version of the file. Server + blockchain compare hashes → Authentic or Tampered.

Tamper Detection
// UPLOAD ANY FILE VERSION — DETECT IF IT WAS MODIFIED
Check File Integrity
ℹ How it works:
1. You upload any version of the file here
2. Node.js computes its SHA256 hash
3. Compared with the original hash stored at upload time
4. Blockchain also independently confirms the result
5. Even one character changed = completely different hash = TAMPERED
